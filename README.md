该项目于2021年3月至2021年5月大学期间学习vue知识时所做项目，目前该项目已不更新

# my-blog
vue实现电商网站

## 安装依赖
```
npm install
```

### 项目启动
```
npm run dev
```

### 打包
```
npm run build
```
